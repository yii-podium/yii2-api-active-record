## Installation

Run console command

```
composer require yii-podium/yii2-api-active-record
```

Or add the package to your `composer.json` file:

```
{
    "require": {
        "yii-podium/yii2-api-active-record": "^0.1"
    }
}
```

then run `composer update`.
